# Hướng Dẫn Sử Dụng a-Shell - Từ Cơ Bản Đến Nâng Cao

Chào mừng bạn đến với bộ tài liệu hướng dẫn sử dụng **a-Shell**. Mục tiêu của thư mục này là giúp bạn khai thác tối đa sức mạnh của chiếc iPhone để lập trình, quản lý hệ thống và tự động hóa công việc.

## Danh Mục Hướng Dẫn

1. **[01_Beginner_Guide.md](./01_Beginner_Guide.md) - Nhập Môn a-Shell**
   - Làm quen với giao diện terminal.
   - Các lệnh Unix cơ bản (`ls`, `cd`, `pwd`).
   - Quản lý file và thư mục trên iOS (`pickFolder`).
   - Tùy biến giao diện (`config`).

2. **[02_Python_Development.md](./02_Python_Development.md) - Lập Trình Python trên iOS**
   - Cài đặt thư viện với `pip`.
   - Chạy script Python.
   - Các giới hạn và cách khắc phục.
   - Sử dụng môi trường ảo.

3. **[03_C_Cpp_Programming.md](./03_C_Cpp_Programming.md) - Lập Trình C/C++ & WebAssembly**
   - Biên dịch mã nguồn C bằng `clang`.
   - Chạy chương trình binary dưới dạng WebAssembly (`wasm`).
   - Liên kết thư viện tĩnh.

4. **[04_Advanced_Features.md](./04_Advanced_Features.md) - Tính Năng Nâng Cao & Scripting**
   - Soạn thảo văn bản với Vim/Nano.
   - Xử lý đa phương tiện với `ffmpeg`.
   - Viết Shell Script tự động hóa.
   - Tương tác với ứng dụng khác qua URL Scheme.

## Mẹo Nhanh
- Gõ `help` để xem danh sách lệnh.
- Gõ `newWindow` để mở thêm cửa sổ terminal mới.
- Vuốt 3 ngón tay để copy/paste (nếu bật trong cài đặt).

Chúc bạn có những trải nghiệm thú vị với a-Shell!
